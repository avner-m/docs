title: 'Events registry'
---

<Tabs>
  <Tab title="Level 1 (Metadata)">
  **Event structure**
    ```
    interface RandomEvent {
    category: "claim-source";
    fieldIncludingPII: string;
    field1: string;
    field2: string;
    timestamp: number;
    type: string;
    version: "1";
    }
    ```
  **Excluded elements**
  ```
  data: {
    error: {
      message: string;
      type: string;
      };
  ```
  </Tab>
  <Tab title="Level 2 (Non-sensitive data)">
  **Event structure**
    ```
    interface RandomEvent {
    category: "claim-source";
    data: {
        error: {
            message: string;
            type: string;
        };
    };
    field1: string;
    field2: string;
    timestamp: number;
    type: string;
    version: "1";
    }
    ```
  **Excluded elements**
    ```
    fieldIncludingPII: string;
    ```
  </Tab>
  <Tab title="Level 3 (Full data)">
    ```
    interface RandomEvent {
    category: "claim-source";
    data: {
        error: {
            message: string;
            type: string;
        };
    };
    fieldIncludingPII: string;
    field1: string;
    field2: string;
    timestamp: number;
    type: string;
    version: "1";
    }
    ```
  </Tab>
</Tabs>
